# spareparts

<!-- badges: start -->

<!-- badges: end -->

**spareparts** is a depot for one-off functions analyzing data through pipelines that will be updated through time. Currently existing pipelines are outlind below:

- Analysis of thermal endurance challenges to measure V02 max (`TEdriftR()`)

## Getting Started

You can install the development version of **spareparts** from
[GitHub](https://github.com/jsalole/spareparts) with `pak()`, or in base R
with `install.packages()`.

``` r
# install.packages("pak")
pak::pak("jsalole/spareparts")
```

``` r
library("spareparts")
```

This will get the package into your R session.
